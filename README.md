# git_guide

### 자주 사용하는 GIT 명령어

```

Branche 생성
- git checkout -b 브랜치명
Branche 푸시 
- git push -u origin 브랜치이름

Merge 취소 
- git merge --abort

Tag 생성 ( 현재 Branche )
- git tag -a v2.0.0 -m "Release version 2.0.0"

Tag 반영 
- git push origin v1.0.1 ( 특정 TAG) 
- git push origin --tags ( 전체 TAG)

Tag 목록
- git ls-remote --tags origin




```
